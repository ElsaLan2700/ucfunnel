# Formik 表單註冊功能

## 使用者操作 SOP

- 請下載本壓縮檔案並解壓縮
- 使用 Visual Studio 開啟壓縮檔的 ucfunnel-main 資料夾
- npm install 安裝 mode_modules
- npm start 啟動專案

## 使用技術

- **React**
  - create react app
  - functional component
- **Redux**
  - store, dispatch, connect
- **Formik**
  - withformik
  - validate
- **Material UI**
